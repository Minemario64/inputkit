# InputKit

Inputkit is a cross-platform python package for getting and handling raw input.
It currently supports windows and posix.

## Setup Instructions to Run / Test Locally

1. Install [uv](https://docs.astral.sh/uv/getting-started/installation/) (if you don't already have it)

2. Clone this repo

3. Go into this project and run `uv sync`

Now you can run the test files
